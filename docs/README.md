understand, integrate, and operate PACT.

| Document | Purpose |

| [AGENT_RUNTIME.md](AGENT_RUNTIME.md) | Controlled runtime boundary, tool allowlists, receipts, and deliverables. |
| [TASK_RECIPES_AND_ARBITRATION.md](TASK_RECIPES_AND_ARBITRATION.md) | Common task categories, acceptance criteria, and arbitration rules. |
| [TRUST_MODEL.md](TRUST_MODEL.md) | Reputation, dispute verdicts, settlement separation, and trust-score updates. |
| [JUDGE_SECURITY.md](JUDGE_SECURITY.md) | Judge hardening, prompt-injection defenses, and review safeguards. |
| [ARC_TESTNET_DEPLOYMENT.md](ARC_TESTNET_DEPLOYMENT.md) | Arc Testnet deployment handoff and required environment variables. |
| [PRODUCTION_READINESS_AUDIT.md](PRODUCTION_READINESS_AUDIT.md) | Production risks, prerequisites, and remaining operational work. |
| [PROJECT_IMPLEMENTATION_AND_RISK_REGISTER.md](PROJECT_IMPLEMENTATION_AND_RISK_REGISTER.md) | Implemented surfaces, unresolved risks, and product boundaries. |

databases, and private deployment notes are intentionally excluded from the
public repository.
